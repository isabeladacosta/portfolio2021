Gallery XR

For this project we worked with the contemporary art gallery Esther Schipper to create a digital tour for one of their art exhibitions. Using Unity I develop an interactive online gallery where the user can walk around, see the art, get information about the artworks and send a request email direct to the gallery. I worked with an UX/UI designer and a 3D artist to make this project come to life. We worked with the agile methodology doing some sprints of 2 weeks to deliver this project.
